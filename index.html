Superstore Profitability Analysis
2025

End-to-end SQL + Excel analysis of 9,800 orders (₹2.26 Cr total sales) to identify why a high-revenue store was losing money. Built a multi-sheet Excel dashboard with pivot tables, slicers, segment look-up tools, and value-tier classification.

Key finding 84.83% of all losses were driven by discounts ≥40%. Tables (−₹1.11L) and Machines (−₹72K) were the top loss sub-categories. Recommended a 30% discount cap — estimated to recover ₹80–90K in margin.
MySQL Excel Pivot Tables CTEs + CASE WHEN
HR Attrition Intelligence Dashboard
2025

Analysed IBM HR dataset to find why employees were leaving. Built SQL queries using CTEs and NTILE window functions, Python EDA with seaborn visualisations, and a 3-page Power BI dashboard with KPI cards, drill-through pages, and DAX measures.

Key finding Sales department had the highest attrition. Employees in the lowest salary quartile were 2.3× more likely to leave. Overtime had a strong positive correlation with exit rate. Recommended salary band review + overtime monitoring programme.
MySQL · CTEs Python · Pandas Power BI · DAX Excel
Café Sales — Power Query Cleaning Pipeline
2025

Built a reusable Power Query (M language) pipeline to transform a 10,001-row dirty café dataset. Handled missing values, enforced data types, standardised inconsistent payment categories, and created a custom Revenue Category column (Low/Medium/High) — all reproducible in one click.

Key outcome Reduced dataset from 10,001 to 9,064 analysis-ready rows. All transformation steps are recorded in the Query Editor — new data with the same format cleans automatically. Demonstrates repeatable, auditable data engineering thinking.
Excel Power Query (M) Pipeline Design
Hotel Booking — Multi-Dataset Excel Cleaning
2025

Cleaned 4 different raw datasets totalling 25,000+ rows using Excel text functions. The hotel booking data arrived as pipe/tilde-delimited strings in a single column ("B001~Rahul Sharma~Delhi~₹4500~Confirmed") and was restructured into an 8-column analysis-ready table using LEFT, RIGHT, MID, FIND, TRIM, and SUBSTITUTE.

Key outcome Applied adaptive formula logic to 4 differently-structured datasets: hotel bookings (1,030 rows), financial transactions (14,039 rows), employee attendance (1,050 rows), and monthly sales report. Demonstrates real-world data wrangling across diverse formats.
Excel Text Functions Data Validation
Brain Stroke Risk Analysis & Prediction
2024

Performed EDA on a healthcare dataset to surface key stroke risk factors. Cleaned data by handling missing values and encoding categoricals. Built and evaluated classification models and analysed feature importance to identify the three highest-risk patient profiles.

Key finding Age, hypertension status, and average glucose level were the top 3 predictors of stroke risk. Patients over 65 with hypertension and glucose levels above 150 mg/dL represented the highest-risk segment, enabling targeted clinical intervention planning.
